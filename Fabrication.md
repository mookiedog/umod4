# Fabrication

Not all of the parts are installed at JLCPCB on the first batch of 4V1 circuit boards.
The following steps are required to complete assembly.

## Install Steps

* Install S1, the EP BOOTSEL switch
* Install J3, the SPI LCD interface socket
* Install D3, the WS2812 RGB status LED used by the WP
* Install J1, the micro SD Card socket
